# Demo Recordings

This folder contains two demo recordings showing how to use Fast-Indexer (cix).

Files:
- `sans-titre.mp4` — short demo clip
- `screen-recording-2026-04-18-184705.mp4` — longer screen recording

Thumbnails:
- `screen-recording-thumb.png` — (optional) thumbnail created from the recording

Usage:
- Play inline from the repository README (GitHub will render `<video>` tags when files are present).
- If the videos are large, they are tracked with Git LFS (`demo/*.mp4`).
